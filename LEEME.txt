Instrucciones para ejecutar el proyecto localmente

Requisitos
Laragon (versión Full)
Versión de PHP utilizada:
El entorno de desarrollo utilizado fue Laragon, el cual incluye una instalación preconfigurada de PHP.
En esta prueba técnica se utilizó la versión PHP 8.3.16 (Win32 vs16 x64).
PostgreSQL instalado y en funcionamiento (17.5-3) 
En la instalación marcar PostgreSQL Server, pgAdmin 4, Command line tools

Instalación de Laragon
Descargar Laragon desde el sitio oficial:
https://laragon.org/download

Descargar específicamente la versión Full desde:
https://github.com/leokhoa/laragon/releases/download/8.2.3/laragon-wamp.exe

Durante la instalación,
desmarcar pretty url
desmarcar las dos casillas finales antes de finalizar el asistente.

Configuración del entorno
Mover el proyecto a la carpeta C:\laragon\www\ (o al directorio donde se haya instalado Laragon).

Abrir Laragon.

Habilitar las extensiones de PostgreSQL:
En el menú de Laragon, ir a: Menu → PHP → php.ini

Buscar (Control + F) y descomentar (eliminar el ; al inicio) las siguientes líneas: 

extension=pgsql
extension=pdo_pgsql
Abrir Laragon e iniciar todos los servicios mediante el botón "Start All".
(Asegurarse de tener marcado Apache)

Configuración de la base de datos
Copiar el archivo config.example.php y renombrarlo a config.php.

Editar el archivo config.php con los parámetros de conexión locales de PostgreSQL.

Asegurarse de que el servidor de PostgreSQL esté activo.

Importar la base de datos ejecutando el script SQL ubicado en:
sql/estructura.sql

Realizar inserts
sql/inserts.sql

Ejecución de la aplicación
Abrir un navegador web.

Acceder al siguiente enlace:
http://localhost/producto-registro-desis/

(En caso de descarga desde github) Acceder al siguiente enlace:
http://localhost/producto-registro-desis-main/
